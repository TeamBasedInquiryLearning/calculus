Section 2.3 Elementary derivative rules (DF3)

Learning Outcomes

  • Compute basic derivatives using algebraic rules.

Observation 2.3.1.
If we recall from the previous section we learned about how to find the derivative of functions using the limit definition of a derivative. From the activities we see that this gets cumbursome when the functions are more complicated. In this section we will discuss a more convenient way to to calculate derivatives.

Definition 2.3.3.

The derivative of the function\(f(x) = x^n\text{,}\) for any nonzero real number \(n\)is

\begin{equation*} f'(x) = nx^{n-1}. \end{equation*}
Observation 2.3.4.
So far we have been using \(f'(x)\) or \(f\) prime to denote a derivative of the function \(f(x)\text{.}\) However there are other ways to denote the derivative of a function. Besides \(f'(x)\) there is also \(\frac{df}{dx}\) or dee-f dee-x. Both notations can be used interchangabley. If you want to take the derivative of \(f'(x)\) or \(\frac{df}{dx}\text{,}\) the notation is \(f''(x)\) or \(\frac{d^2f}{dx^2}\text{;}\) this is known as second derivative of \(f(x)\text{.}\)

Definition 2.3.6.

If \(f(x) = c\text{,}\) for some constant \(c\in\mathbb{R}\text{,}\) then \(f'(x) = 0\text{.}\)

Definition 2.3.7.

The Constant Multiple Rule: If \(c\) is a constant and \(f(x)\) is a differentiable function, then

\begin{equation*} \frac{d}{dx}\left[cf(x)\right] = c\frac{d}{dx}\left[f(x)\right] \end{equation*}
Definition 2.3.8.

The Sum(Difference) Rule: If \(f(x)\) and \(g(x)\) are both differentiable, then

\begin{equation*} \frac{d}{dx}\left[f(x) \pm g(x)\right] = \frac{d}{dx}\left[f(x)\right]\pm\frac{d}{dx}\left[g(x)\right] \end{equation*}
