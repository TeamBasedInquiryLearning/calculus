Activity 2.3.11.

What are the first and second derivatives for the arbitrary parabola given by \(f(x) = ax^2 + bx + c\text{,}\) where \(a,\,b,\,c\) are any real numbers and \(a \neq 0\text{?}\)

  1. \(\displaystyle f'(x) = 2ax + bx + c, \, f''(x)=2a +b.\)

  2. \(\displaystyle f'(x) = 2x + 1, \, f''(x)=2.\)

  3. \(\displaystyle f'(x) = 2ax + b , \, f''(x)=2a.\)

  4. \(\displaystyle f'(x) = ax + b, \, f''(x)=a.\)
