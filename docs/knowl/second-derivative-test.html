Theorem 3.5.10. The Second Derivative Test.

Suppose that \(x=c\) is a critical point of \(f(x)\) and that \(f''(x)\) is continuous at \(x=c\text{.}\)

  1. If \(f'(c) = 0\) and \(f''(c) < 0 \text{,}\) then a relative maximum occurs when \(x=c\text{.}\)

  2. If \(f'(c) = 0\) and \(f''(c) > 0 \text{,}\) then a relative minimum occurs when \(x=c\text{.}\)

  3. If \(f'(c) = 0\) and \(f''(c) = 0 \text{,}\) then this test is inconclusive.

In case of an inconclusive Seconf Derivative Test, we need to go back to the First Derivative Test to classify the critical point.
