Activity 2.7.6.

To take the derivative of some explicit equations you might need to make it an implicit equation. For this activity we will find the derivative of \(y = x^x\text{.}\) Make the equation an implicit equation by taking natural logarithm of both sides, this gives \(\ln(y) = x\ln(x)\text{.}\) Knowing this, what is \(\frac{dy}{dx}\text{?}\)

  1. \(\displaystyle \frac{dy}{dx} = x^x(\ln(x) + 1)\)

  2. \(\displaystyle \frac{dy}{dx} = \frac{(\ln(x)+1)}{x^x}\)

  3. \(\displaystyle \frac{dy}{dx} = x^x(\ln(x) + x)\)

  4. \(\displaystyle \frac{dy}{dx} = \frac{(\ln(x)+x)}{x^x}\)
