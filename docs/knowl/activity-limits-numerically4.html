Activity 1.2.8.

Consider the following function \(f(x)= 3x^3 + 2x^2- 5x+20.\)

(a)

Of the following options, which of the 5 values would be best to help you find \(\lim_{x \to 2}f(x)\) numerically?

  1. 1.9, 1.99, 2.0, 2.01, 2.1

  2. 1.98, 1.99, 2.0, 2.01, 2.02

  3. 1.8, 1.9, 2.0, 2.1, 2.2

  4. 1.0, 1.5, 2.0, 2.5, 3.0

(b)

Use the values that you chose in part A to calculate an approximation for \(\lim_{x \to 2}f(x)\text{.}\)

(c)

Which value best describes the limit that you obtained in part (b)?

  1. the approximate value is 41.25

  2. the approximate value is 41.5

  3. approximate value is 41.75

  4. approximate value is 42
