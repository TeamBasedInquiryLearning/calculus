Activity 2.4.7.

Just like with the product rule, there are times where we can find the derivative of a quotient using elementary rules rather than the quotient rule. Which of the following quotients could you find the derivative of without using the quotient rule? Select all that apply.

  1. \(\displaystyle f(x) = \frac{6}{x^3}\)

  2. \(\displaystyle f(x) = \frac{2}{\ln x}\)

  3. \(\displaystyle f(x) = \frac{e^x}{\sin x}\)

  4. \(\displaystyle f(x) = \frac{x^3+3x}{x}\)
