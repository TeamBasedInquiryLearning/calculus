Activity 4.9.10.

Suppose we wanted to try substitution method to find \(\int e^x\cos(e^x+3)\,dx\text{.}\) Which of these choices for \(u\) appears to be most useful?

  1. \(u=x\text{,}\) so \(du=dx\)

  2. \(u=e^x\text{,}\) so \(du=e^x\,dx\)

  3. \(u=e^x+3\text{,}\) so \(du=e^x\,dx\)

  4. \(u=\cos(x)\text{,}\) so \(du=-\sin(x)\,dx\)

  5. \(u=\cos(e^x+3)\text{,}\) so \(du=e^x\sin(e^x+3)\,dx\)
