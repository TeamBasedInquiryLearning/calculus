Section 1.2 Limits numerically (LT2)

Learning Outcomes

  • Infer the value of a limit based on nearby values of the function.

Activity 1.2.1.

Based on the values of Table 1.2.2, what is the best approximation for \(\lim_{x\to 7}\text{?}\)

  1. the limit is approximately 7

  2. the limit is approximately 0.17

  3. the limit is approximately 0.16

  4. the limit is approximately 0.1667

  5. the limit is approximately 6.9999

Remark 1.2.3.

Notice that the value we obtained in Activity 2 is only an approximation, based on the trends that we have seen within the table.

Activity 1.2.4.

In Activity 1.1.1 we got an approximation to the limit of the function as x tends to 2. Now let us say you are given the same plot as in Activity 1.1.1, but also given a table of numerical values (Table 1.2.5) for the function. Given this new information what is the limit of the function as x tends to 2?

  1. There is not enough information because we do not know the value of the function at \(x = 2\)

  2. Can be approximated to be 1 because the data in the table and the graph show that from the left and the right the function approaches 1 as \(x\) goes to 2

  3. Can be approximated to be 1 because the values appaer to approach 1 and the graph appears to approach 1, but we should zoom in on the graph to be sure

  4. Cannot be approximates because the function might not exist at \(x = 2\text{.}\)

Activity 1.2.6.

Based on Table 1.2.7, what information can be inferred about \(\lim_{x\to 1}f(x)\text{,}\) \(\lim_{x\to 1^-}\)f(x), and \(\lim_{x\to 1^+}f(x)\text{?}\)

  1. \(\lim_{x\to 1^-}f(x) = -0.5,\,\, \lim_{x\rightarrow 1^+}f(x) = 0.5\text{,}\) and \(\lim_{x\rightarrow 1}f(x) = 0\)

  2. \(\lim_{x\to 1^-}f(x) = -0.5,\,\, \lim_{x\rightarrow 1^+}f(x) = 0.5\text{,}\) and \(\lim_{x\rightarrow 1}f(x)\) does not exist

  3. \(\lim_{x\rightarrow 1^-}f(x) = 0.5,\,\, \lim_{x\rightarrow 1^+}f(x) = -0.5\text{,}\) and \(\lim_{x\rightarrow 1}f(x)\) does not exist

  4. \(\lim_{x\rightarrow 1^-}f(x) = 0.5,\,\, \lim_{x\rightarrow 1^+}f(x) = -0.5\text{,}\) and \(\lim_{x\rightarrow 1}f(x) = 0\)

Activity 1.2.8.

Consider the following function \(f(x)= 3x^3 + 2x^2- 5x+20.\)

  1. Choose 6 values of x that you think will help you find \(\lim_{x \to 2}f(x)\) numerically.

  2. Use the values that you chose in part A to calculate an approximation for \(\lim_{x \to 2}f(x)\text{.}\)

  3. Which value best describes the limit that you obtained in part B?
