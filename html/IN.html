Chapter 4 Definite and Indefinite Integrals (IN)

Learning Outcomes

By the end of this chapter, you should be able to...

  1. Use geometric formulas to compute definite integrals.

  2. Approximate definite integrals using Riemann sums.

  3. Determine basic antiderivatives.

  4. Solve basic initial value problems.

  5. Evaluate a definite integral using the Fundamental Theorem of Calculus.

  6. Find the derivative of an integral using the Fundamental Theorem of Calculus.

  7. Use definite integrals to find area under a curve.

  8. Use definite integral(s) to compute the area bounded by several curves.

Readiness Assurance.

Before beginning this chapter, you should be able to...

  1. Find the derivative of a function using elementary derivative rules, as in Section 2.3. (CheckIt 1 )

  2. Use sigma notation for sums. (Khan Academy 2  and Active Calculus 3 )

  3. Find the area of a rectangle in the coordinate plane. (Example at Khan Academy 4 Example at virtual nerd 5 )

  4. Find the intersection of two graphs. (Khan Academy 6 )

  5. Find the area of plane shapes, such as rectangles, triangles, circles, and trapezoids. (Math is fun 7 )
