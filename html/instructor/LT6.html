Section 1.6 Limits with infinite outputs (LT6)

Learning Outcomes

  • Determine limits of functions approaching vertical asymptotes.

  5. The function has a jump in outputs around \(x=0\)

Definition 1.6.2.

A function has a vertical asymptote at \(x=a\) when

\begin{equation*} \lim_{x\to a} f(x) = + \infty \end{equation*}

or

\begin{equation*} \lim_{x\to a} f(x) = - \infty \end{equation*}

The limit being equal to positive infinity means that we can make the output of \(f(x)\) as large a positive number as we want as long as we are sufficiently close to \(x=a\text{.}\) Similarly, the limit being equal to negative infinity means that we can make the output of \(f(x)\) as large a negative number as we want as long as we are sufficiently close to \(x=a\text{.}\)

Activity 1.6.3.

Which of the following functions have vertical asymptotes? Select all!

Remark 1.6.4.

If \(x=a\) is a vertical asymptote for the function \(f(x) \text{,}\) the function \(f(x)\) is not defined at \(x=a\text{.}\) As \(f(a)\) does not exist, the function is NOT continuous at \(x=a\text{.}\) Moreover, the function's output tends to plus or minus infinity and so the limit is not equal to a number.

Activity 1.6.5.

Notice that as \(x\) goes to 0, the value of \(x^2\) goes to 0 but the value of \(1/x^2\) goes to infinity. What is the best explanation for this behavior?

  1. When dividing by an increasingly small number we get an increasing big number

  2. When dividing by an increasingly large number we get an increasing small number

  3. A rational function always has a vertical asymptote

  4. A rational function always has a horizontal asymptote

Remark 1.6.6.

Informally, we say that the limit of "\(\frac{1}{0}\)" is infinite. Notice that this could be either positive or negative infinity, depending on how whether the outputs are becoming more and more positive or more and more negative as we approach zero.

Remark 1.6.8.

When considering a ratio of functions \(f(x)/g(x) \text{,}\) the inputs \(a\) where \(g(a)=0\) are not in the domain of the ratio. If \(g(a)=0 \) but \(f(a) \) is not equal to 0, then \(x=a\) is a vertical asymptote

Activity 1.6.9.

Consider the function \(f(x)=\frac{x^2-1}{x-1}\text{.}\) The line \(x=1\) is NOT a vertical asymptote for \(f(x)\text{.}\) Why?

  1. When \(x\) is not equal to \(1\text{,}\) we can simplify the fraction to \(x-1\text{,}\) so the limit is \(1\text{.}\)

  2. When \(x\) is not equal to \(1\text{,}\) we can simplify the fraction to \(x+1\text{,}\) so the limit is \(2\text{.}\)

  3. The function is always equal to \(x+1\text{.}\)

  4. The function is always equal to \(x-1\text{.}\)

Activity 1.6.12.

Explain and demonstrate how to find the value of each limit.

  1. \begin{equation*} \lim_{x\to-3^- } \frac{{\left(x + 4\right)}^{2} {\left(x - 2\right)}}{{\left(x + 3\right)} {\left(x - 5\right)}} \end{equation*}
  2. \begin{equation*} \lim_{x\to-3^+ } \frac{{\left(x + 4\right)}^{2} {\left(x - 2\right)}}{{\left(x + 3\right)} {\left(x - 5\right)}} \end{equation*}
  3. \begin{equation*} \lim_{x\to-3 } \frac{{\left(x + 4\right)}^{2} {\left(x - 2\right)}}{{\left(x + 3\right)} {\left(x - 5\right)}} \end{equation*}
Answer.
  1. \begin{equation*} \lim_{x\to-3^- } \frac{{\left(x + 4\right)}^{2} {\left(x - 2\right)}}{{\left(x + 3\right)} {\left(x - 5\right)}}=-\infty \end{equation*}
  2. \begin{equation*} \lim_{x\to-3^+ } \frac{{\left(x + 4\right)}^{2} {\left(x - 2\right)}}{{\left(x + 3\right)} {\left(x - 5\right)}}=+\infty \end{equation*}
  3. \begin{equation*} \lim_{x\to-3 } \frac{{\left(x + 4\right)}^{2} {\left(x - 2\right)}}{{\left(x + 3\right)} {\left(x - 5\right)}} \text{ does not exist } \end{equation*}
