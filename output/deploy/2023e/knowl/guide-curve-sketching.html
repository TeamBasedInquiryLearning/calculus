Remark 3.7.7. A guide to curve sketching.

  1. Identify the domain of the function.
  2. Identify any vertical or horizontal asymptotes, if they exist.
  3. Find \(f'(x)\text{.}\) Then use it to determine the intervals where the function is increasing and the intervals where the function is decreasing. State any local extrema.
  4. Find \(f''(x)\text{.}\) Then use it to determine the intervals where the function is concave up and the intervals where the function is concave down. State any inflection points.
  5. Put everything together and draw sketch.
