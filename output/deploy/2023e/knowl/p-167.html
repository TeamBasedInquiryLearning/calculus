Let \(f\) and \(g\) be functions defined on an open interval \(I\) containing the real number \(c\) satisfying
\begin{equation*} \lim_{x \to c} f(x) = L \, \text{and} \, \lim_{x \to c} g(x) = K, \end{equation*}
for \(L\) and \(K\) some real numbers. Then we have the following limits.
  1. Constant Law: \(\displaystyle \lim_{x \to c} b = b\text{,}\) for \(b\) any constant real number;
  2. Identity Law: \(\displaystyle \lim_{x \to c} x = c\) ;
  3. Sum/Difference Law: \(\displaystyle \lim_{x \to c} (f(x) \pm g(x)) = L \pm K\text{;}\)
  4. Scalar Multiple Law: \(\displaystyle \lim_{x \to c} b \cdot f(x) = bL\text{,}\) for \(b\) any constant real number;
  5. Product Law: \(\displaystyle \lim_{x \to c} f(x) \cdot g(x) = LK\text{;}\)
  6. Quotient Law: if \(K \ne 0\text{,}\) then \(\displaystyle \lim_{x \to c} f(x) / g(x) = L/K\text{.}\)
