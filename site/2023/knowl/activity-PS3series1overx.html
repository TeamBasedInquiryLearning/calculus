Activity 9.3.4.

Suppose we wish to find the power series for the function \(f(x)=\frac{1}{x}\text{.}\)
(a)
Which of the following represents the power series for \(g(r)=\frac{1}{1-r}\text{?}\)
  1. \(g(r)=\displaystyle\sum_{n=0}^\infty rx^n\text{.}\)
  2. \(g(r)=\displaystyle\sum_{n=0}^\infty (rx)^n\text{.}\)
  3. \(g(r)=\displaystyle\sum_{n=0}^\infty r^n\text{.}\)
  4. \(g(r)=\displaystyle\sum_{r=0}^\infty x^r\text{.}\)
(b)
For what value of \(r\) is \(\frac{1}{1-r}=\frac{1}{x}\text{?}\)
  1. \(r=x-1\text{.}\)
  2. \(r=1-x\text{.}\)
  3. \(r=x+1\text{.}\)
  4. \(r=-x\text{.}\)
(c)
Substituting \(r\) with this value, which of the following is a power series for \(f(x)=\frac{1}{x}\text{?}\)
  1. \(f(x)=\displaystyle\sum_{n=0}^\infty (-x)^n\text{.}\)
  2. \(f(x)=\displaystyle\sum_{n=0}^\infty (1-x)^n\text{.}\)
  3. \(f(x)=\displaystyle\sum_{n=0}^\infty (x-1)^n\text{.}\)
  4. \(f(x)=\displaystyle\sum_{n=0}^\infty (1+x)^n\text{.}\)
(d)
Given that the domain of convergence for \(r\) in \(f(r)\) is \(-1 < r < 1\text{,}\) what should be the domain of convergence for \(x\) in \(f(x)\text{?}\)
  1. \(-1 < x < 1\text{.}\)
  2. \(-2 < x < 0\text{.}\)
  3. \(-2 < x < 2\text{.}\)
  4. \(0 < x < 2\text{.}\)
