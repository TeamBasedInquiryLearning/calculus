An example of a lemniscate is given in FigureÂ 51. The equation of this lemniscate is \((x^{2} + y^{2})^2 = x^2 - y^2\text{.}\) What is the derivative with respect \(x\) for this lemniscate? (Solve for \(\frac{dy}{dx}\)).
  1. \(\displaystyle \frac{dy}{dx} = \frac{x(1-2(x^2+y^2))}{y+2(x^2+y^2)}\)
  2. \(\displaystyle \frac{dy}{dx} = \frac{x(1-2(x^2+y^2))}{y(1+2(x^2+y^2))}\)
  3. \(\displaystyle \frac{dy}{dx} = \frac{y(1+2(x^2+y^2))}{x(1-2(x^2+y^2))}\)
  4. \(\displaystyle \frac{dy}{dx} = \frac{y+2(x^2+y^2)}{x(1-2(x^2+y^2))}\)
