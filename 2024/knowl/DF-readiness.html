Readiness Assurance.

Before beginning this chapter, you should be able to...
  1. Write equations of lines using slope-intercept and/or point-slope form (Math is Fun 1 )
  2. Find the average rate of change of a function over some interval (Khan Academy 2 )
  3. Evaluate functions at variable expressions (Purple Math 3 )
  4. Use the laws of exponents to rewrite a given expression (Khan Academy (1) 4  and Khan Academy (2) 5 )
  5. Compose and decompose functions (Khan Academy 6 )
  6. Recall special trig values on the unit circle (Khan Academy (1) 7  and Khan Academy (2) 8 )
