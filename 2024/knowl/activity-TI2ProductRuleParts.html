Activity 5.2.1.

Answer the following.
(a)
Using the product rule, which of these is derivative of \(x^3e^x\) with respect to \(x\text{?}\)
  1. \(\displaystyle 3x^2e^x\)
  2. \(\displaystyle 3x^2e^{x}+x^3e^x\)
  3. \(\displaystyle 3x^2e^{x-1}\)
  4. \(\displaystyle \frac{1}{4}x^4 e^x\)
(b)
Based on this result, which of these would you suspect to equal \(\int 3x^2e^x+x^3e^x\,dx\text{?}\)
  1. \(\displaystyle x^3e^x+C\)
  2. \(\displaystyle x^3e^x+\frac{1}{4}x^4e^x+C\)
  3. \(\displaystyle 6xe^x+3x^2e^x+C\)
  4. \(\displaystyle 6xe^x+3x^2e^x+3x^2e^x+x^3e^x+C\)
