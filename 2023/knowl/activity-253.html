Activity 5.1.1.

Answer the following.
(a)
Using the chain rule, which of these is the derivative of \(e^{x^3}\) with respect to \(x\text{?}\)
  1. \(\displaystyle e^{3x^2}\)
  2. \(\displaystyle x^3e^{x^3-1}\)
  3. \(\displaystyle 3x^2e^{x^3}\)
  4. \(\displaystyle \frac{1}{4}e^{x^4}\)
(b)
Based on this result, which of these would you suspect to equal \(\int x^2e^{x^3}\,dx\text{?}\)
  1. \(\displaystyle e^{x^3+1}+C\)
  2. \(\displaystyle \frac{1}{3x}e^{x^3+1}+C\)
  3. \(\displaystyle 3e^{x^3}+C\)
  4. \(\displaystyle \frac{1}{3}e^{x^3}+C\)
