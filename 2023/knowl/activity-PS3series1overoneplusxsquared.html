Activity 9.3.10.

Suppose we wish to find the power series for the function \(f(x)=\frac{1}{1+x^2}\text{.}\) Recall that \(g(x)=\frac{1}{1-r}=\displaystyle\sum_{n=0}^\infty r^n.\)

(a)

For what value of \(r\) is \(\frac{1}{1-r}=\frac{1}{1+x^2}\text{?}\)

  1. \(r=x^2\text{.}\)

  2. \(r=-x^2\text{.}\)

  3. \(r=1-x^2\text{.}\)

  4. \(r=x^2-1\text{.}\)

(b)

Evaluating \(r\) at the previously found value, which of the following is the power series of \(f(x)=\frac{1}{1+x^2}\text{?}\)

  1. \(\displaystyle\frac{1}{1+x^2}=\sum_{n=0}^\infty (-1)^nx^{2n}\text{.}\)

  2. \(\displaystyle\frac{1}{1+x^2}=\sum_{n=0}^\infty (1-x^2)^n\text{.}\)

  3. \(\displaystyle\frac{1}{1+x^2}=\sum_{n=0}^\infty x^{2n}\text{.}\)

  4. \(\displaystyle\frac{1}{1+x^2}=\sum_{n=0}^\infty (x^2-1)^n\text{.}\)

(c)

Given that the interval of convergence for \(r\) is \(-1 < r < 1\text{,}\) what is the interval of convergence for \(x\text{?}\)

  1. \(-1 < x < 1\text{.}\)

  2. \(-1 < x < 0\text{.}\)

  3. \(0 < x < 1\text{.}\)

  4. \(0 < x < 4\text{.}\)

(d)

How can the power series for \(\frac{1}{1+x^2}\) be manipulated to obtain a power series for \(\arctan(x)\text{?}\)

  1. Differentiate each term.

  2. Integrate each term.

  3. Replace \(x\) with \(x^2\) in each term.

  4. Replace \(x\) with \(1/x\) in each term.

(e)

Which of these power series is the result of this manipulation?

  1. \(\displaystyle\arctan(x)=\sum_{n=0}^\infty (-1)^n\frac{x^{2n+1}}{2n+1}\text{.}\)

  2. \(\displaystyle\arctan(x)=\sum_{n=0}^\infty (-1)^n\frac{x^{2n-1}}{2n-1}\text{.}\)

  3. \(\displaystyle\arctan(x)=\sum_{n=0}^\infty (-1)^n(2n)x^{2n-1}\text{.}\)

  4. \(\displaystyle\arctan(x)=\sum_{n=0}^\infty (-1)^n(2n+1)x^{2n}\text{.}\)
