Activity 5.2.3.

Answer the following.
(a)
Which of the following equations is equivalent to the formula \(\frac{d}{dx}[uv]=u'v+uv'\text{?}\)
  1. \(\displaystyle uv'=-\frac{d}{dx}(uv)-vu'\)
  2. \(\displaystyle uv'=-\frac{d}{dx}(uv)+vu'\)
  3. \(\displaystyle uv'=\frac{d}{dx}(uv)+vu'\)
  4. \(\displaystyle uv'=\frac{d}{dx}(uv)-vu'\)
(b)
Which of these is the most concise result of integrating both sides with respect to \(x\text{?}\)
  1. \(\displaystyle \int(uv')\,dx=uv-\int(vu')\,dx\)
  2. \(\displaystyle \int(u)\,dv=uv-\int(v)\,du\)
  3. \(\displaystyle \int(uv')\,dx=uv-\int(vu')\,dx+C\)
  4. \(\displaystyle \int(u)\,dv=uv-\int(v)\,du+C\)
