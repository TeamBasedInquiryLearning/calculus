Chapter 5 Techniques of Integration (TI)

Learning Outcomes

How do we use various techniques to integrate less simple functions?

By the end of this chapter, you should be able to...

  1. Evaluate various integrals via the substitution method.

  2. Compute integrals using integration by parts.

  3. Compute integrals involving products of trigonometric functions.

  4. Use trigonometric substitution to compute indefinite integrals.

  5. I can integrate functions using a table of integrals.

  6. I can integrate functions using the method of partial fractions.

  7. I can select appropriate strategies for integration.

  8. I can compute improper integrals.

Readiness Assurance.

Before beginning this chapter, you should be able to...

  1. Recognize when to apply the chain rule, and use it successfully.

    • Review: Khan Academy 1 

  2. Recognize when to apply the product rule, and use it successfully.

    • Review: Khan Academy 2 

  3. Perform basic definite integrals.

    • Review: Khan Academy 3 

  4. Compute basic antiderivatives.

    • Review: Khan Academy 4 

  5. Differentiate trigonometric functions.

    • Review: Khan Academy 5 

  6. Compute limits.

    • Review: Khan Academy 6 

    • Review: Khan Academy 7 

  7. Apply the pythagorean identity.

    • Review: Khan Academy 8
