Section 3.4 Extreme values (AD4)

Learning Outcomes

  • Use the Extreme Value Theorem to find the absolute maximum and minimum values of a continuous function on a closed interval.

Remark 3.4.1.

In many different settings, we are interested in knowing where a function achieves its least and greatest values. These can be important in applications—say to identify a point at which maximum profit or minimum cost occurs—or in theory to characterize the behavior of a function or a family of related functions.

Example 3.4.2.

Consider the simple and familiar example of a parabolic function such as \(s(t) = -16t^2 + 32t + 48\) that represents the height of an object tossed vertically: its maximum value occurs at the vertex of the parabola and represents the greatest height the object reaches. This maximum value is an especially important point on the graph, the point at which the curve changes from increasing to decreasing.

Definition 3.4.3.

Given a function \(f\text{,}\) we say that \(f(c)\) is a global or absolute maximum of \(f\) provided that \(f(c) \ge f(x)\) for all \(x\) in the domain of \(f\text{,}\) and similarly we call \(f(c)\) a global or absolute minimum of \(f\) whenever \(f(c) \le f(x)\) for all \(x\) in the domain of \(f\text{.}\)

The absolute maxima and minima are also called the absolute extrema or extreme values of the function.

Observation 3.4.5.

From Activity 3.4.4 we notice that there are some issues when determining the absolute mimimum and maximum values of a function from its graph alone. The Extreme Value Theorem will guarantee the existence of extrema on a closed interval. Thne we will learn about critical points to be able to detect extrema algebraically.

Theorem 3.4.6.

If \(f\) is continuous on a closed interval \([a,b]\text{,}\) then \(f\) has both an absolute maximum and an absolute minimum on the interval.

Definition 3.4.10.

We say that \(x\) is a critical point (or critical number) of \(f(x)\) if \(x\) is in the domain of \(f(x)\) and either \(f'(x) = 0\) or \(f'(x)\) does not exist.

Activity 3.4.11.

  1. \(x = \sqrt{2}\) and \(x = -\sqrt{2}\text{.}\)

  2. \(x = \sqrt{2}\text{.}\)

  3. \(x = 2\) and \(x = 0\text{.}\)

  4. \(x = 2\text{.}\)

Remark 3.4.12. The Closed Interval Method.

The following is a way of finding the absolute extrema of a continuous \(f\) on a closed interval.

  1. Make a list of all critical points of \(f\) in \((a,b)\text{.}\) (Do not include any critical points outside of the interval).

  2. Add the endpoints \(a\) and \(b\) to the list.

  3. Evaluate \(f\) at all points on your list.

  4. The smallest value is the absolute minimum. The largest is the absolute maximum.

Activity 3.4.15.

Explain how to find the absolute minimum and absolute maximum values of the function \(f(x)=-2 \, x^{3} + 18 \, x^{2} + 42 \, x + 33\) on the interval \([-2,2]\text{.}\)

Answer.

The absolute extrema may occur at these \(x\)-values: \(-2, -1, 2\text{.}\)

  • \(\displaystyle f(-2)=37\)

  • \(\displaystyle f(-1)=11\)

  • \(\displaystyle f(2)=173\)

Thus the absolute minimum is \(11\) and the absolute maximum is \(173\text{.}\)

(The value \(f(7)=523\) is ignored as \(7\) isn't in the given interval.)
