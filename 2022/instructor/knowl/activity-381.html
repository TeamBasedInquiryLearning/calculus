Activity 6.2.1.

Suppose we wanted to find the arclength of the parabola \(y=-x^2+6x\) over the interval \([0,4]\text{.}\)

(a)

Suppose we wished to estimate this length with two line segments, where \(\Delta x=2\text{.}\)

Which of the following expressions represents the sum of the lengths of the line segments with endpoints \((0,0)\text{,}\) \((2,8)\) and \((4,8)\text{.}\)

  1. \(\sqrt{4+8}\text{.}\)

  2. \(\sqrt{4^2+8^2}\text{.}\)

  3. \(\sqrt{2^2+8^2}+\sqrt{4^2+8^2}\text{.}\)

  4. \(\sqrt{2^2+8^2}+\sqrt{(4-2)^2+(8-8)^2}\text{.}\)

(b)

Suppose we wished to estimate this length with four line segments, where \(\Delta x=1\text{.}\)

Which of the following expressions represents the sum of the lengths of the line segments with endpoints \((0,0)\text{,}\) \((1,5)\text{,}\) \((2,8)\text{,}\) \((3,9)\) and \((4,8)\text{.}\)

  1. \(\sqrt{4^2+8^2}\text{.}\)

  2. \(\sqrt{1^2+(5-0)^2}+\sqrt{1^2+(8-5)^2}+\sqrt{1^2+(9-8)^2}+\sqrt{1^2+(8-9)^2}\text{.}\)

  3. \(\sqrt{1^2+5^2}+\sqrt{2^2+8^2}+\sqrt{3^2+9^2}+\sqrt{4^2+8^2}\text{.}\)

(c)

Suppose we wished to estimate this length with \(n\) line segments \(\Delta x=4/n\text{.}\) Let \(f(x)=-x^2+6x\text{.}\)

Which of the following expressions represents the length of the line segment from \((x_0, f(x_0))\) to \((x_0+\Delta x, f(x_0+\Delta x))\text{?}\)

  1. \(\sqrt{x_0^2+f(x_0)^2}\text{.}\)

  2. \(\sqrt{(x_0+\Delta x)^2+f(x_0+\Delta x)^2}\text{.}\)

  3. \(\sqrt{(\Delta x)^2+(f(x_0+\Delta x)-f(x_0))^2}\text{.}\)

  4. \(\sqrt{(\Delta x)^2+f(\Delta x)^2}\text{.}\)

(d)

Which of the following Riemann sums best estimates the arclength of the parabola \(y=-x^2+6x\) over the interval \([0,4]\text{?}\) Let \(f(x)=-x^2+6x\text{.}\)

  1. \(\displaystyle \sum \sqrt{x_i^2+f(x_i)^2}\text{.}\)

  2. \(\displaystyle \sum \sqrt{(x_i+\Delta x)^2+f(x_i+\Delta x)^2}\text{.}\)

  3. \(\displaystyle \sum \sqrt{(\Delta x)^2+(f(x_i+\Delta x)-f(x_i))^2}\text{.}\)

  4. \(\displaystyle \sum \sqrt{(\Delta x)^2+f(\Delta x)^2}\text{.}\)

(e)

Note that

\begin{align*} \sqrt{(\Delta x)^2+(f(x_i+\Delta x)-f(x_i))^2} & = \sqrt{(\Delta x)^2\left(1+\left(\frac{f(x_i+\Delta x)-f(x_i)}{\Delta x} \right)^2\right)}\\ &=\sqrt{1+\left(\frac{f(x_i+\Delta x)-f(x_i)}{\Delta x} \right)^2}\Delta x \end{align*}

Which of the following best describes \(\displaystyle\lim_{\Delta x\to 0} \frac{f(x_i+\Delta x)-f(x_i)}{\Delta x}\text{?}\)

  1. \(0\text{.}\)

  2. \(1\text{.}\)

  3. \(f'(x_i)\text{.}\)

  4. This limit is undefined.
