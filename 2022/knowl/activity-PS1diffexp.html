Activity 9.1.5.

(a)

Given \(f(x)=\displaystyle \sum_{n=0}^\infty \frac{1}{n!}x^n\text{,}\) which of the following best represents \(f'(x)=\frac{d}{dx}\left[\displaystyle \sum_{n=0}^\infty \frac{1}{n!}x^n\right]\text{?}\)

  1. \(\displaystyle \sum_{n=0}^\infty \frac{1}{n!}x^n\text{.}\)

  2. \(\displaystyle \sum_{n=0}^\infty \frac{1}{n!}x^{n-1}\text{.}\)

  3. \(\displaystyle n\sum_{n=0}^\infty \frac{1}{n!}x^{n-1}\text{.}\)

  4. \(\displaystyle n\sum_{n=0}^\infty \frac{1}{(n-1)!}x^{n-1}\text{.}\)

  5. \(\displaystyle n\sum_{n=1}^\infty \frac{1}{(n-1)!}x^{n-1}\text{.}\)

(b)

For which of the following is \(y=\frac{dy}{dx}\text{?}\)

  1. \(y=x^2\text{.}\)

  2. \(y=\sqrt{x}\text{.}\)

  3. \(y=e^x\text{.}\)

  4. \(y=\ln(x)\text{.}\)

  5. \(y=\sin(x)\text{.}\)

  6. \(y=\cos(x)\text{.}\)
