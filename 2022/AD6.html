Remark 3.6.3.

On the leftmost curve in Figure 51, as we move from left to right, the slopes of those tangent lines will increase. Therefore, the rate of change of the pictured function is increasing, and this explains why we say this function is increasing at an increasing rate.

Observation 3.6.4.

We must be extra careful with our language when dealing with negative numbers. For example, it can be tempting to say that “\(-100\) is bigger than \(-2\text{.}\)” But we must remember that “greater than” describes how numbers lie on a number line: \(-100\) is less than \(-2\) becomes it comes earlier on the number line. It might be helpful to say that “\(-100\) is "more negative" than \(-2\text{.}\)”

Remark 3.6.6.

We encountered before the language of concavity: when a curve bends upward, we say its shape is concave up. When a curve bends downwards, we say its shape is concave down.

Activity 3.6.7.

Which curve is concave up? Which one is concave down? Why? Try to explain using the graph!

Definition 3.6.8.

Let \(f\) be a differentiable function on an interval \((a,b)\text{.}\) Then \(f\) is concave up on \((a,b)\) if and only if \(f'\) is increasing on \((a,b)\text{;}\) \(f\) is concave down on \((a,b)\) if and only if \(f'\) is decreasing on \((a,b)\text{.}\)

Theorem 3.6.10. Test for Concavity.

Suppose that \(f(x)\) is twice differentiable on some interval \((a,b)\text{.}\) If \(f'' > 0\) on \((a,b)\text{,}\) then \(f\) is concave up on \((a,b)\text{.}\) If \(f'' < 0\) on \((a,b)\text{,}\) then \(f\) is concave down on \((a,b)\text{.}\)

Observation 3.6.11.

In the previous section, we saw in Activity 3.5.8 how to use critical points of the function and the values of the first derivative to identify intervals of increase/decrease of a function. The next activity Activity 3.6.12 uses the critical points of the derivative function and the values of the second derivative as in Theorem 3.6.10 to identify where the original function is concave up/down.

(b)

(c)

(d)

(e)

Definition 3.6.13.

If \(x=c\) is a point where \(f''(x)\) changes sign, then the concavity of graph of \(f(x)\) changes and we call \(x=c\) an inflection point of \(f(x)\text{.}\)

Activity 3.6.14.

Activity 3.6.15.

For each of the following functions, describe the open intervals where it is concave up or concave down, and any inflection points.

  1. \(\displaystyle f(x)=-\frac{1}{4} \, x^{5} - \frac{5}{2} \, x^{4} - \frac{15}{2} \, x^{3}\)

  2. \(\displaystyle f(x)=\frac{3}{20} \, x^{5} + x^{4} - \frac{5}{2} \, x^{3}\)

Answer.

  1. \(\displaystyle f''(x)=-5 \, {\left(x + 3\right)}^{2} x\)

    • \((-\infty, -3 )\) is concave down

    • \(-3\) is not an inflection point

    • \(( -3 , 0 )\) is concave down

    • \(0\) is an inflection point

    • \(( 0, \infty )\) is concave up

  2. \(\displaystyle f''(x)=3 \, {\left(x + 5\right)} {\left(x - 1\right)} x\)

    • \((-\infty, -5 )\) is concave down

    • \(-5\) is an inflection point

    • \(( -5 , 0 )\) is concave up

    • \(0\) is an inflection point

    • \(( 0 , 1 )\) is concave down

    • \(1\) is an inflection point

    • \(( 1 , \infty )\) is concave up
